Role: Project Manager
Le project manager est le responsable de l’ensemble de l’unité Data Science d’une entreprise. Une unité de science des données peut avoir plusieurs équipes, chacune travaillant sur plusieurs projets de science des données dans des secteurs d'activité distincts. Un responsable de groupe peut déléguer ses tâches à un substitut, mais les tâches associées au rôle ne changent pas.
Relationships
Primary Performs
  • Create project charter
  • Finalize documentation
Modifies
  • Project Charter
  • Project final report
    Main Description

    Le project manager exécute les tâches suivantes:

    Créer un compte de groupe sur une plateforme d'hébergement de code (telle que GitHub, Git, VSTS ou autres)
    Créez un référentiel de modèles de projet sur le compte de groupe et générez-le à partir du référentiel de modèles de projet développé, ainsi qu'un ensemble de modèles de documents normalisés pour guider un processus efficace de data science.
    Créez un référentiel d’utilitaires. Le référentiel d'utilitaires fournit un ensemble d'utilitaires utiles pour rendre le travail d'un data scientist plus efficace, notamment des utilitaires d'exploration, d'analyse et de création de rapports interactifs, ainsi que de modélisation et de création de rapports de base.
    Configurez la stratégie de contrôle de la sécurité de ces deux référentiels sur votre compte de groupe.
