Role: Solution architect
Les solution architect définissent la manière dont les données seront stockées, utilisées, intégrées et gérées par différentes entités de données et systèmes informatiques, ainsi que par toute application utilisant ou traitant ces données d'une manière ou d'une autre.

    Main Description

    Solution architect exécute les tâches suivantes:

    Organise les données au niveau macro.
    Organise les données au niveau micro, les modèles de données, pour une nouvelle application.
    Fournit un modèle de données logique en tant que norme pour la source d'or et pour les applications consommatrices à hériter.
    Fournit un modèle de données logique avec les éléments et les règles métier nécessaires à la création de règles de qualité des données (DQ).
