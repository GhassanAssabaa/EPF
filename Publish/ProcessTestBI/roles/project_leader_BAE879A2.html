Role: Project lead

Un chef de projet gère les activités quotidiennes de chaque scientifique sur un projet spécifique.
Relationships
Primary Performs
  • Checkpoint project
  • Create template repository
  • Provision data infrastructure
  • Transition to production support
Main Description

Le chef de projet exécute les tâches suivantes :

Créez un référentiel de projet sous le projet d'équipe et générez-le à partir du référentiel de modèles de projet Team.
Créez un stockage de fichiers Azure à utiliser pour stocker les actifs de données du projet.
Montez le stockage de fichiers Azure sur la machine virtuelle Data Science (DSVM) et ajoutez-y des actifs de données de projet.
Configurez le contrôle de sécurité en ajoutant des membres du projet et configurez leurs privilèges.
